## Pipeline

1. **Run the helper script first**:
   ```
   python3 scripts/build_lifestyle_json.py sunday --dry-run > /tmp/sun-raw.json
   ```
   Review the JSON. Helper already populated weather, calendar, mail
   (48h weekend recap), portfolio, curated travel list, STL events.

2. **Enrich with qualitative fields**:
   - `pillars.weather.whatToWear` — one sentence on what to wear
     for the day.
   - `pillars.life.oneThingToPlan` — forward-looking planning nudge
     (schedule a coffee, block focus time Wed afternoon, sign up for
     the thing you keep meaning to). Different from Saturday's
     present-tense "do this today."
   - `pillars.life.localPicksNextWeekend` — filter STL events to
     next-Sat/Sun window; keep 1–2.
   - `pillars.life.rec` — weekend read (longread / book chapter /
     podcast episode for the week ahead).
   - `pillars.retirement.weekEndReflection` — quiet week-end
     reflection (not a planning nudge).

3. **Filter drive-distance ideas** in `pillars.vacation.driveDistanceIdeas`
   to 2–4 framed for "good for next weekend."

4. **Write the enriched JSON** to `out/lifestyle/<date>.json`.

5. **Ship it** — single helper call handles git push AND Vercel deploy:
   ```
   python3 scripts/build_lifestyle_json.py sunday --skip-deploy --date <date>
   ```
   `--skip-deploy` is critical here: the helper still does the manifest
   refresh + git commit + git push (the parts that finish in <2s), but
   skips the Vercel deploy (which can take 30-60s and risk the 3-min
   hard interrupt on cron sessions).

   After that returns 0, trigger the Vercel deploy in a separate fast
   background step:
   ```
   nohup bash -c 'cd /Volumes/BotCentral/Users/milo/repos/Milo/website && npm run fetch-briefs && vercel deploy --prod --yes' > /tmp/sun-deploy.log 2>&1 &
   ```
   `nohup ... &` detaches the deploy from the cron session so the 3-min
   hard interrupt on this prompt doesn't kill it mid-deploy. The Vercel
   build will keep running and finish on its own.

6. **Final response** — one-line confirmation:
   ```
   Published Sunday Lifestyle Brief for <date>: https://daily-brief-tau.vercel.app/weekend/<date>
   ```
   (The site may take ~60s to reflect the deploy; that's expected.)
